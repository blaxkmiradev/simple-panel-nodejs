## 🛠 Quick Start

### 1. Clone & Install

```bash
# Clone the repository
git clone https://github.com/blaxkmiradev/simple-panel-nodejs.git
cd simple-panel-nodejs

# Install dependencies
npm install express express-session
```

### 2. Launch

```bash
node index.js
```

### 3. Access

Open your browser and navigate to:

> 👉 **http://localhost:3000**

**Default credentials:**

| Field | Value |
|---|---|
| Username | `Mira` |
| Password | `Nika` |
